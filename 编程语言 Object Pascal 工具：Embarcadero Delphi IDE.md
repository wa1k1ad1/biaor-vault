###### ✅窗口程序“点击按钮A，显示/隐藏欢迎”
**按钮，变化控制**
```
procedure TForm1.按钮1Click(Sender: TObject);
begin
  // 切换文本的显示/隐藏状态
  文本框1.Visible := not 文本框1.Visible;  // True相变False
  
  // 根据状态更新按钮文字
  if 文本1.Visible then
    按钮1.Caption := '隐藏文本'
  else
    按钮1.Caption := '显示文本';
end;
```
解析：
- **条件判断**：根据文本的显示状态动态改变按钮文字
- **用户体验**：按钮文字明确指示下次点击的效果
这样实现后，按钮就会在"显示文本"和"隐藏文本"之间切换，同时控制文本的显示和隐藏状态！
