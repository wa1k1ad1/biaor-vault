**创作认知**：
- 内容创作不应依赖灵感，也不应从零开始写作
- 利用已有笔记和资料，避免重复思考，利用积累，节省时间精力
- 采用CODE法则：CO（发散）捕捉组织资料，DE（聚合）萃取表达内容

**创作误区**：
- ❌ 从 0 开始写（重复思考，浪费积累时间精力）%%没有具体的主题，想到什么写什么%%
- ❌ 等待💡灵感出现（持续的创作，💡灵感不可靠）

**创作原则**：
- 围绕一个项目进行CO和DE，项目的完成，其实是CO发散 和 DE聚合收敛的过程
	- CO"搜索相关的想法、素材、观点"，收集到足够条件切换DE聚合模式 ^7843f9
- "不要选择从零开始"，利用过去积累的结晶（前期建立可重复的工作流）
