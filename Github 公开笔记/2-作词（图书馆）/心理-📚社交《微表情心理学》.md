- **微表情心理学**是心理学的一个分支，主要研究人类在极短时间内（通常为 **1/25 秒至 0.5 秒**）闪现的、不受意识控制的细微面部表情，以及这些表情背后的真实情绪和心理状态。  
- 

###### **1. 微表情的核心概念**
- **定义**：微表情是人在试图隐藏真实情绪时，短暂、无意识流露的真实表情，通常一闪而过，难以伪装。  
- **与普通表情的区别**：
  - **持续时间极短**（普通表情可持续数秒，微表情仅0.04~0.5秒）。  
  - **不受主观控制**（即使刻意掩饰，真实情绪仍可能通过微表情泄露）。  
  - **全球通用性**（不同文化背景的人，基本情绪如愤怒、恐惧、厌恶的微表情表现相似）。  

---
###### **2. 微表情的常见类型**
根据心理学家 **保罗·艾克曼（Paul Ekman）**（《别对我撒谎》原型人物）的研究，人类有 **7 种基本情绪**对应的微表情：  

| **情绪** | **典型微表情特征** | **应用场景** |
|----------|------------------|--------------|
| **高兴** | 眼角皱纹（鱼尾纹）、嘴角上扬（真笑 vs 假笑） | 判断对方是否真心愉悦 |
| **愤怒** | 眉毛下压、嘴唇紧闭或咬牙切齿 | 识别潜在的敌意或不满 |
| **悲伤** | 眉头内侧上扬、嘴角下拉 | 发现隐藏的悲伤或抑郁 |
| **恐惧** | 眉毛上扬并聚拢、眼睛睁大 | 测谎、危机预警 |
| **厌恶** | 鼻子微皱、上唇抬起 | 察觉反感或轻蔑 |
| **惊讶** | 眉毛大幅上扬、瞳孔扩大 | 判断是否真实震惊 |
| **轻蔑** | 单侧嘴角上扬（不对称微笑） | 识别傲慢或不屑 |

---
###### **3. 微表情的应用领域**
- **测谎与刑侦**：FBI、警方审讯时用于识别嫌疑人是否说谎。  
- **商业谈判**：观察对方真实态度，调整谈判策略。  
- **人际关系**：判断他人是否真诚，避免被欺骗或PUA。  
- **心理学研究**：用于情绪识别、自闭症诊断等。  

---
###### **4. 如何学习识别微表情？**
1. **观察关键区域**：眉毛、眼睛、嘴角、鼻翼的变化。  
2. **对比基线行为**：先了解对方正常表情，再捕捉异常瞬间。  
3. **结合语境分析**：微表情需结合语言、肢体动作综合判断。  
4. **训练工具**：可通过 **METT（微表情训练工具）** 或 **FACS（面部动作编码系统）** 练习。
