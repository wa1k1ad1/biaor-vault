###### 核心概念输入
- **卢曼卡片盒本质**：以标准化格式记录互相关联的笔记，通过连接构建知识网络，让系统自下而上地生长
- **核心价值**：帮助思考、连接想法、让知识自然生长的"外接大脑"
- **核心理念**：建立笔记间的连接，让想法在对话中涌现

###### 四种卡片类型
1. 💡**灵感笔记**：捕捉转瞬即逝的想法
2. 📜**文献笔记**：阅读过程的共鸣记录
3. 🔏**永久笔记**：萃取有价值内容，建立连接
4. 📍**索引笔记**：汇总主题相关的知识卡片

##### 输出
选择输出方式：Obsidian实践方案

###### Obsidian实践流程
**输入阶段**：
- 灰文件管理：在输入文件夹集中存放相关内容
- 使用双链连接，不做过早分类
- 启用**Zettelkasten navigation**插件生成树状结构导图

**扩充阶段**：
- 文本颗粒化管理：[[0 闪念笔记（💡灵感库）]]
- 局部关系图辅助整理

**提炼阶段**：
- 从灵感/文献笔记中萃取永久笔记
- 建立笔记间的深度连接
- 避免过度分类，注重内容分析

###### 表达输出方案
**卡片分类体系**：
- 💡灵感笔记 → 个人想法思考
- 📜文献笔记 → 外部资料摘录  
- 🔏永久笔记 → 个人思考提炼
- ⛑项目笔记 → 行动相关材料

**存放策略**：
- 基于近期使用价值分类
- 与相关项目笔记建立索引
- 设置预期调用时间
