## 🧠 SuperMemo 18 极简入门四步曲

### 🧩 第一步：创建你的知识库 - “集合”

1. **打开 SuperMemo 18。**
2. **新建集合：** 点击 `文件(File)` -> `新建(New)` -> `知识库(Knowledge System)` 。
3. **取个名字：** 比如 “英语词汇”、“考研政治”、“个人知识库”，选好保存位置，点 `保存` 。这个文件 (`.kno` 后缀) 就是你所有记忆内容的家了。

### 📝 第二步：添加要记忆的内容 - “条目”和“问答”

这是最核心的一步！你要把想记住的东西做成 **问答卡片** 。

- **问 (Question)：** 提示信息，触发你回忆。
- **答 (Answer)：** 你要回忆出的具体内容。

#### 举个栗子🌰 - 背英语单词

- **问：** `苹果 (中文意思)`
- **答：** `apple`

#### 如何添加：

1. 确保打开了你的“集合”。
2. **按 `Alt + A` (最常用！)** 。这是添加新条目的快捷键。
3. **弹出窗口：**
	- **问题(Question)：** 在第一个大文本框里输入你的“问”，比如 `苹果 (中文意思)` 。
	- **答案(Answer)：** 在下面的大文本框里输入你的“答”，比如 `apple` 。
	- **标题(Title)：** 可以自动生成（就是问题的前几个字），也可以手动改个简短标题，比如 `apple` 。标题主要用于浏览查找。
4. **点 `确定` (或按 `Enter`)** 。恭喜！你的第一张记忆卡片诞生了！🎉
5. **重复 `Alt + A` ：** 继续添加你想记的其他单词或知识点。

#### 📌 新手友好小贴士

- **从简单的开始：** 先加 5-10 个卡片练手，别贪多。
- **问题要清晰：** “问” 的部分要能明确指向“答”。避免模糊的提问。
- **答案要简洁：** 尽量用关键词或短句。太长的答案复习起来累。
- **活用回车：** 在问题和答案文本框里，按 `Enter` 可以换行，适合答案分点。

### 🔁 第三步：最重要的环节 - 复习 (学习)

这是 SuperMemo 发挥魔力的时刻！它会根据算法，在 **最佳时间** 弹出你需要复习的卡片。

#### 如何开始复习：

1. **打开你的集合。**
2. **按 `Ctrl + L` (Learn 的意思)** 。这是开始或继续学习的快捷键。 **这是你每天最常用的键！**
3. **弹出卡片：** SuperMemo 会显示一张卡片的 **“问”** 部分。

#### 复习时的操作：

1. **努力回忆答案！** 看着“问”（比如 `苹果 (中文意思)` ），在脑子里想答案是什么 (`apple`)。
2. **显示答案：**
	- 想好后， **按 `空格键`** 。这时完整的“答”部分 (`apple`) 会显示出来。
3. **自我评分：** 这是最关键的一步！根据回忆的难易程度，告诉 SuperMemo：
	- **`忘记` (0分 - Bad)：** 完全没想起来，或者答错了。按键盘 **`0`** 。
	- **`困难` (3分 - Hard)：** 想起来了，但很费劲、不确定。按键盘 **`3`** 。
	- **`良好` (4分 - Good)：** 稍微想了一下就想起来了，比较顺畅。按键盘 **`4`** 。 **(最常用的评分！)**
	- **`简单` (5分 - Easy)：** 秒答，太简单了。按键盘 **`5`** 。
	- *(也有1,2分，介于忘记和困难之间，新手可以先主要用 0,3,4,5)*
4. **自动安排下次：** 根据你的评分，SuperMemo 会自动计算出这张卡片下次什么时候再出现。评分越高（记得越牢），下次出现间隔越长。
5. **下一张卡片：** 评分后，SuperMemo 会自动显示下一张需要复习卡片的“问”。重复步骤 1-4。
6. **结束复习：** 当今天需要复习的卡片都处理完了，或者你想暂停时，按 `Esc` 键退出学习模式。

#### 📌 复习核心原则

- **诚实评分！** 这是算法准确的基础。别欺骗自己。
- **回忆是关键！** 一定要先 **努力想** ，再看答案。
- **“良好” (4分) 是主力：** 大部分顺畅回忆的卡片评 4 分。
- **每天坚持：** **每天按 `Ctrl + L` 复习！** 哪怕只有5分钟。中断会让积累的“快要忘记”的卡片堆积如山，体验变差。养成习惯最重要！

### 📚 第四步：查看与管理 - 内容窗口

- **按 `Ctrl + Alt + W`** ：打开“内容”窗口。这里像一本书的目录，可以看到你添加的所有条目（卡片）。
- **浏览：** 可以上下滚动查看。
- **查找：** 在顶部搜索框输入关键词（标题或内容里的字）查找条目。
- **编辑：** 双击一个条目，可以修改它的“问”和“答”。
- **删除：** 选中条目，按 `Delete` 键（谨慎使用！）。

## 🌟 给新手的超级重要建议

1. **第一天目标：** 安装好 SM18 中文版 -> 创建一个集合 -> 添加 5-10 张简单卡片 -> 按 `Ctrl + L` 复习一遍。 **这就成功了！**
2. **坚持一周：** 头几天可能觉得有点陌生，坚持每天打开，添加少量新卡片（比如每天5个），并按 `Ctrl + L` 复习。几天后你就能感受到它提醒你复习“快要忘记”东西的神奇之处。
3. **核心快捷键记牢：**
	- `Alt + A` ： **添加** 新卡片
	- `Ctrl + L` ： **学习/复习** (每天必按！)
	- `空格` ： 显示答案
	- `0, 3, 4, 5` ： 评分 (尤其 `4`)
	- `Ctrl + Alt + W` ： 打开内容窗口
4. **别怕犯错：** 评分错了、卡片做的不完美都没关系！可以随时编辑修改。先动起来最重要。
5. **循序渐进：** 不要一上来就研究模板、插件、图片、挖空等高级功能。 **先把纯文字问答卡片的添加和复习流程玩熟练！** 这是根基。
6. **拥抱“未知”：** 刚开始，你不知道卡片下次什么时候出现，完全信任算法就好。这正是 SuperMemo 自动化的优势。
7. **资源求助：**
	- 安装包通常自带详细中文说明书 (PDF)。
	- 搜索 “\[SuperMemo 18 中文 教程\]”。
	- 关注“褪墨”、“学习骇客”、“阳志平的网志”等博客/公众号，有很多深入文章。
	- 加入相关的论坛或社群（如知乎圈子、QQ群等）提问。

## 📅 总结：新手每日操作清单

1. **打开** SuperMemo 18 和你的知识库集合。
2. **添加新卡：** 按几次 `Alt + A` ，输入你想记的新内容 (每天少量，比如 5-10 个)。
3. **复习旧卡：** 按 `Ctrl + L` ，开始复习。
	- 看到“问”，努力想答案。
	- 按 `空格` 显示答案。
	- 按 `0/3/4/5` 诚实评分。
	- 重复直到今日复习结束 (按 `Esc` 退出)。
4. **(可选)** 按 `Ctrl + Alt + W` 浏览或编辑你的卡片。

**坚持几天，你就会发现：那些你评了高分（4/5）的卡片，真的隔了很久再出现，而且你还记得！这就是间隔重复的魅力！** 入门后，再去探索更强大的功能（图片、音频、挖空测试、模板等）会让你的记忆如虎添翼。祝你学习高效，记忆长久！ 💪🏻
